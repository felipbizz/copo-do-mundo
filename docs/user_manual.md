# Manual do Usuário

## Introdução

O Copo do Mundo é uma aplicação web para gerenciar competições de bebidas. Este manual explica como usar o sistema para votar e visualizar resultados.

## Acesso ao Sistema

### Iniciando a Aplicação
1. Abra o navegador
2. Acesse: http://localhost:8501
3. Aguarde o carregamento da interface

### Identificação
- Não é necessário login
- Cada voto é anônimo
- Um código único é gerado para rastreamento

## Interface de Votação

### Seleção de Bebidas
1. Escolha uma bebida da lista
2. Visualize a imagem e descrição
3. Avalie os critérios

### Critérios de Avaliação
- **Originalidade**: Criatividade da receita
- **Aparência**: Apresentação visual
- **Sabor**: Qualidade do gosto

### Sistema de Pontuação
- Nota de 1 a 5 estrelas
- Comentários opcionais
- Confirmação antes de enviar

### Envio de Votos
1. Preencha todos os critérios
2. Revise suas avaliações
3. Clique em "Enviar Voto"

## Recursos Adicionais

### Rascunhos
- Salvamento automático
- Recuperação de votos não enviados
- Edição de votos pendentes

### Progresso
- Indicador de votos restantes
- Histórico de votos
- Status de envio

### Resultados
- Visualização em tempo real
- Gráficos interativos
- Comparação entre bebidas

## Dicas e Truques

### Atalhos
- `Tab`: Navegação entre campos
- `Enter`: Envio de formulário
- `Esc`: Cancelar operação

### Navegação
- Use as abas para alternar entre seções
- Role para ver mais opções
- Clique em ícones para ajuda

### Visualização
- Ajuste o zoom do navegador
- Use modo tela cheia
- Alternar tema claro/escuro

## Solução de Problemas

### Problemas Comuns
1. **Página não carrega**
   - Verifique a conexão
   - Recarregue a página
   - Limpe o cache

2. **Voto não enviado**
   - Verifique campos obrigatórios
   - Tente novamente
   - Salve como rascunho

3. **Imagens não aparecem**
   - Verifique conexão
   - Recarregue a página
   - Use modo offline

### Mensagens de Erro
- Leia a mensagem com atenção
- Siga as instruções
- Registre o erro se persistir

### Recuperação
- Use o botão "Voltar"
- Recupere rascunhos
- Reinicie a sessão
